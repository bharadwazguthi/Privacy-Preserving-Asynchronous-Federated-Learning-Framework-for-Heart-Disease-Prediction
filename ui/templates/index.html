A Novel Asynchronous Federated Learning Framework for Enhanced Prediction of Cardiovascular Diseases

(AFLCP)

warning

Problem Statement

Cardiovascular diseases (CVDs) are a leading cause of death globally. Early and accurate prediction of CVDs is crucial for timely intervention and improved patient outcomes. Traditional machine learning models often rely on centralized datasets, raising privacy concerns and limiting access to diverse patient data. Federated learning (FL) offers a promising solution by enabling collaborative model training across decentralized datasets without sharing sensitive patient information. However, existing synchronous FL approaches can be inefficient due to varying data availability and computational capabilities across participating institutions.

check_circle

Objectives

  • Design and implement an asynchronous FL algorithm that accommodates varying data availability and computational resources across institutions.
  • Integrate advanced AI techniques, such as deep learning models, to improve the accuracy and robustness of CVD prediction.
  • Evaluate the performance of AFLCP against existing synchronous FL methods such as FedAvg and FedProx using real-world datasets.
  • Develop a user-friendly web application for visualizing results and insights.

settings Algorithms

FedAvg: A synchronous FL algorithm where the server aggregates model updates from all clients at each round.

FedProx: An extension of FedAvg that addresses statistical heterogeneity by adding a proximal term.

AFLCP: Our proposed asynchronous algorithm that allows clients to update the global model independently, improving efficiency.
