Global Federated Model

This page provides access to the final global model produced by the Asynchronous Federated Learning with Communication Preservation (AFLCP) framework. The model is trained collaboratively across multiple simulated healthcare clients without sharing raw patient data.

Usage Notes

  • The global model can be deployed for inference without retraining.
  • All preprocessing artifacts (scaler, feature mapping) must be used together with the model.
  • Model training follows privacy-preserving federated learning principles.
